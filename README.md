## Running app locally

1. Clone this repo:

   ```
   $ git clone https://github.com/martinsik/bsc-angular-demo
   ```
   
1. Install dependencies:

   ```
   $ cd bsc-angular-demo
   $ npm i
   ```

1. Run tests:

   ```
   $ ng test
   ```
   
1. Start development server:

   ```
   $ ng serve
   ```
